Serverless micropub

Provides a serverless function for micropub -> hugo markdown

Getting Started

You can clone the repo and adjust the following settings in config.json

  • ghUser your github username
  • sites the site specific configuration

NOTE: Your micropub endpoint will be available at /api/[site]. This means that if you take the configuration as is, the URL will be /api/main

You will also need to add a personal access token as an environment variable named GH_TOKEN
